# CSS Grid 布局指南

本文介绍 CSS Grid 的使用方法。

## 基本概念

Grid 布局由两个核心概念组成：
- **Grid Container**: 网格容器
- **Grid Item**: 网格项

## 简单示例

```css
.container {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    gap: 1rem;
}
```
